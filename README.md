# Pharma GraphRAG Pilot

A pilot project exploring Graph-based Retrieval-Augmented Generation (GraphRAG) for pharmaceutical use cases using Neo4j.

## Overview

This project leverages Neo4j's graph database capabilities combined with RAG techniques to enable intelligent querying and reasoning over pharmaceutical data.

## Tech Stack

- **Neo4j** — Graph database for storing and querying pharma knowledge graphs
- **GraphRAG** — Retrieval-Augmented Generation over graph-structured data
